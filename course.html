Chapitre 2.4 — Les types de données en Python

En Python, chaque valeur a un type. Le type détermine ce que tu peux faire avec cette valeur. Python détecte automatiquement le type.

Concept cléPython est un langage à typage dynamique : le type est déterminé automatiquement par la valeur.

Les 4 types de base

# 1. Entier (int) age = 25 # 2. Décimal (float) prix = 19.99 # 3. Texte (str) nom = "Marie" # 4. Booléen (bool) est_connecte = True

Vérifier le type

La fonction type() te dit le type d'une variable :

print(type(age)) # <class 'int'> print(type(prix)) # <class 'float'> print(type(nom)) # <class 'str'>
AstuceDans VS Code, survole une variable avec la souris — l'éditeur affiche son type automatiquement.

Convertir les types

age_texte = "25" age_nombre = int(age_texte) # str → int prix_int = int(19.99) # Donne 19 texte = str(25) # Donne "25"
