Commençons par le tri à bulles. Le principe est simple : on compare deux éléments voisins et on les échange si nécessaire. On répète jusqu'à ce que tout soit trié.

Voici le code complet :
# Tri à bulles — Python def tri_a_bulles(liste): n = len(liste) for i in range(n): for j in range(0, n-i-1): if liste[j] > liste[j+1]: liste[j], liste[j+1] = liste[j+1], liste[j] return liste # Test nombres = [64, 34, 25, 12, 22] print(tri_a_bulles(nombres)) # Résultat : [12, 22, 25, 34, 64]
La complexité est O(n²) — simple à comprendre, mais pas le plus efficace pour les grandes listes.
